What is Agent Catalog?

Your Single Source of Truth for AI Agent Governance

Agent Catalog is the central registry and governance platform for AI agents and MCP servers. It provides visibility into your agent ecosystem, enabling informed decisions at plan time rather than runtime.

  • ✓ Discover and browse agents & MCP servers
  • ✓ Track agent reputation and performance metrics
  • ✓ Understand sub-agent relationships and dependencies
  • ✓ Access best practices and invocation examples
  • ✓ Manage versioning, evals, and lifecycle
